You can download the jar files and put them on your classpath.

However we recommend you use a dependency management system such as Maven so you do not have to manage dependencies of dependencies (such as the Ebean dependencies) yourself. See the Maven dependencies here.

Otherwise you can download the jars from these links:

  • Ebean ORM 3.2.4 [ jar | src | doc ]
  • Java Persistence API 1.0.2 (Java annotations needed to markup your entities) [ jar ]
  • Ebean bytecode enhancement agent 3.2.2 [ jar ]
  • Ebean bytecode agent loader 1.1.1 [ jar ]
